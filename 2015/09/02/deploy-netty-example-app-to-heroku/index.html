本地测试

用mvn clean install打包到target目录下，然后运行heroku local web,测试应用在本地是否工作正常。

发布和部署

  1. 用heroku create命令新建一个应用。
  2. 本地提交，并将代码提交到heroku，git push heroku master。
  3. heroku ps:scale web=1保证至少有一个实例在运行这个应用。
  4. heroku open可以打开部署后的服务页面。
